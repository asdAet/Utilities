# DuplicateLineCleaner

## Что делает скрипт

Скрипт удаляет повторяющиеся строки из одного текстового файла. Он обновляет исходный файл на месте и оставляет только первые уникальные вхождения строк.

## Что нужно изменить перед запуском

Открой [main.py](./main.py) и проверь переменную:

- `file_path` — полный путь к файлу, который нужно очистить от дублей.

## Как пользоваться

1. Открой [main.py](./main.py).
2. Укажи нужный путь в `file_path`.
3. Запусти скрипт:

```powershell
python main.py
```

## Что получится после запуска

Исходный файл будет обновлён:

- все повторяющиеся строки исчезнут;
- останется только первое появление каждой строки;
- порядок первых уникальных строк сохранится.

В консоли появится сообщение об успешной обработке или ошибке.

## Как работает внутри

Скрипт не переписывает файл "вживую" во время чтения. Вместо этого он делает безопаснее:

1. Берёт путь к исходному файлу.
2. Создаёт временный файл с суффиксом `.tmp`.
3. Читает исходный файл построчно.
4. Хранит уже встреченные строки во множестве `seen_lines`.
5. Если строка встречается впервые, записывает её во временный файл.
6. После завершения заменяет исходный файл временным через `os.replace()`.

## Что считается дублем

Дубликат определяется по полному совпадению строки. Это значит:

- строка с лишним пробелом уже считается другой;
- строка с другим регистром тоже считается другой;
- разный символ конца строки тоже влияет на совпадение.

## Что получает на вход

- один текстовый файл;
- файл должен быть доступен на чтение и запись;
- кодировка ожидается UTF-8.

## Что создаёт на выходе

- обновлённый исходный файл без повторяющихся строк.

## Особенности этого скрипта

- работает только с одним файлом за запуск;
- использует временный файл, а не прямую перезапись во время чтения;
- не делает отдельную резервную копию;
- все настройки задаются прямо в коде.

## Когда использовать

- когда есть один длинный список с повторами;
- когда нужно быстро очистить файл без ручной обработки;
- когда важно сохранить порядок первых уникальных строк.
